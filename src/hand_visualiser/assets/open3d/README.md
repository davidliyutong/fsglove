# Open3D Directory

This directory contains configuration files for Open3D visualization library.

## Files

- **viewpoint.json** - Saved camera viewpoint configuration for Open3D visualizer
  - Stores camera position, orientation, and rendering parameters
  - Used to restore consistent viewing angles across visualization sessions

## Purpose

This directory stores Open3D-specific configuration and state files that help maintain consistent visualization settings and camera perspectives when working with 3D hand models and point clouds.
